- 개발 때 테스트
  1. makefile 만들어 놨으니
  2. make front로 프론트 서버 띄우고(백그라운드 아님)
  3. make dev 로 (venv환경 설정 하고) 백엔드 서버 띄워서(백그라운드 아님)
  4. http://localhost:3000 으로 접속해서 테스트 가능

- 현재 컨테이너는 백엔드 서버 + 벡터디비 마운트해서 사용(빌드 할 때 오래 걸릴까봐 미리 세팅된 벡터디비 사용)
  - 백엔드 서버 실행 후 첫 분석은 모델 로드하느라 좀 느림
  - 임베딩하는건 app/
